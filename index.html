AWS CI/CD Pipeline Demo

Automatic Website Deployment using AWS Cloud Services

About This Project

This project demonstrates Continuous Integration and Continuous Deployment (CI/CD) using AWS services. When code is pushed to GitHub, AWS CodePipeline automatically deploys the updated website to Amazon S3.

AWS Services Used

CodePipeline

Automates the entire CI/CD workflow from GitHub to deployment.

CodeBuild

Builds and prepares the application before deployment.

Amazon S3

Hosts the static website and delivers it globally.
